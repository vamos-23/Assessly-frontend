# Assessly – Exam Portal

## 📌 Overview
Assessly is a full-stack web application designed to streamline the management of exams and user roles. It provides a secure authentication system and a scalable backend architecture to handle exam-related operations efficiently.

---

## 🌐 Live Demo
**Frontend (Vercel):**  
https://assessly-mu.vercel.app

---

## 🚀 Features
- Secure user authentication using JWT  
- Role-based access control (e.g., student/admin)  
- RESTful API architecture using Spring Boot  
- Responsive and user-friendly interface  
- Integration with cloud-hosted PostgreSQL database  

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)  
- Deployed on Vercel  

### Backend
- Spring Boot  
- REST APIs  
- Deployed on Render  

### Database
- PostgreSQL (Neon)  

### Authentication
- JSON Web Tokens (JWT)  

---

## ⚙️ Setup Instructions

### Backend
1. Navigate to `assessly-backend`  
2. Configure database credentials in `application.properties`  
3. Run the Spring Boot application  

### Frontend
1. Navigate to `assessly-frontend`  
2. Install dependencies:  
   ```bash
   npm install
   ```  
3. Start the development server:  
   ```bash
   npm start
   ```
